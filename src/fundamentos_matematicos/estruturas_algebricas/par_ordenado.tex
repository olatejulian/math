    \begin{defi}[Par Ordenado]\label{defi:par-ordenado}
        É um par de elementos a qual a ordem de ocorrência de cada elemento é significante. Geralmente representado por $(a,b)$. Seja $a$ e $b$ elementos quaisquer.
    \end{defi}
